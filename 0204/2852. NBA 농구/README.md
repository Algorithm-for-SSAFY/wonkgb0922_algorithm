### 문제 설명

<p>동혁이는 NBA 농구 경기를 즐겨 본다. 동혁이는 골이 들어갈 때 마다 골이 들어간 시간과 팀을 적는 이상한 취미를 가지고 있다.

농구 경기는 정확히 48분동안 진행된다. 각 팀이 몇 분동안 이기고 있었는지 출력하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 골이 들어간 횟수 N(1<=N<=100)이 주어진다. 둘째 줄부터 N개의 줄에 득점 정보가 주어진다. 득점 정보는 득점한 팀의 번호와 득점한 시간으로 이루어져 있다. 팀 번호는 1 또는 2이다. 득점한 시간은 MM:SS(분:초) 형식이며, 분과 초가 한자리 일 경우 첫째자리가 0이다. 분은 0보다 크거나 같고, 47보다 작거나 같으며, 초는 0보다 크거나 같고, 59보다 작거나 같다. 득점 시간이 겹치는 경우는 없다.</p>

### 출력 

 <p>첫째 줄에 1번 팀이 이기고 있던 시간, 둘째 줄에 2번 팀이 이기고 있던 시간을 출력한다. 시간은 입력과 같은 형식(MM:SS)으로 출력한다.</p>

### 💡접근 방법
> 깡 구현문제이다. 문자열 자체보다 문자열을 파싱해서 `int`로 변환해서 풀었다.
> 현재 이기는 팀과 시간이 들어오면, 우선 승수를 비교한다. 1번팀이 이기고 있다면 이전 시간과 막 들어온 시간의 차를 1번 팀의 분과 초에 누적시킨다. 2번팀이 이기고 있다면 마찬가지. 그리고 승수를 카운팅한다.
> 마지막은 48분이므로, 48분과 비교하여준다. 그리고 지금까지 시간을 누적했을 때, 초의 경우 올리거나 내려야 하므로 60으로 나눈 값을 분에 더해주고, 60의 나머지값을 취한다.
> 여기서 초가 음수가 될 경우, 분에서 1을 빼어 초에 60을 더해주는 것으로 양수로 만들 수 있다.
> 마지막에는 `printf()`를 이용하여 형식에 맞게 출력해주었다.
