### 문제 설명

<p>방향 없는 그래프가 주어졌을 때, 연결 요소 (Connected Component)의 개수를 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 정점의 개수 N과 간선의 개수 M이 주어진다. (1 ≤ N ≤ 1,000, 0 ≤ M ≤ N×(N-1)/2) 둘째 줄부터 M개의 줄에 간선의 양 끝점 u와 v가 주어진다. (1 ≤ u, v ≤ N, u ≠ v) 같은 간선은 한 번만 주어진다.</p>

### 출력 

 <p>첫째 줄에 연결 요소의 개수를 출력한다.</p>

### 💡접근 방법
> dfs로 풀었다. 정점과 연결된 다른 노드를 저장하기 위해 `List`를 배열로 선언하였고, 이 `List`가 `null`일 경우 생성자로 초기화하는 코드도 만들어주었다.
> 무방향 그래프이므로 자신과 연결된 다른 정점에도 자신을 추가해주어야 한다.
> `visited` 배열을 통해 방문 여부를 확인해주었고, 간선이 없는 경우 탐색하지 않는 대신 카운팅을 해주었다. 자기 자신도 카운트 되므로.
